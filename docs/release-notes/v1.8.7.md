# CFST-GUI v1.8.7

## 变更摘要

- 修复 Windows 安装前 WebView2 Runtime 检测：缺失运行时时先提示并安装依赖，避免安装后首次启动直接失败。
- 增加任务历史保留和诊断导出能力，便于定位长期运行、后台恢复和用户环境差异导致的问题。
- 修复 Android post-probe 收尾等待流程，任务结束后会等待移动端后处理完成，减少状态提前返回和资源清理竞态。
- Release workflow 恢复完整发行链路：GitHub Release 成功发布后自动调用 GHCR workflow，产出 `linux/amd64` 与 `linux/arm64` 多架构镜像并更新 `latest`。
- Release、Android resubmit、Container workflow、Wails、Go runtime 和 Android 默认版本号统一提升到 `1.8.7`，Android 默认 `versionCode` 提升到 `10807`。
