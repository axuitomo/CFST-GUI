# CFST-GUI v1.8.4

## 变更摘要

- 前端包管理迁移到 pnpm workspace：新增根目录 `package.json`、`pnpm-workspace.yaml`、`.npmrc` 和 `pnpm-lock.yaml`，移除 `frontend/package-lock.json`，CI、脚本、文档和 Release workflow 全部改用 pnpm。
- 在线更新链路收口为直连 GitHub Releases latest + manifest：更新检查、manifest 读取和安装包下载均禁用环境代理，并通过 GitHub 加速候选链并发竞速后使用首个通过 SHA256 校验的结果。
- 新增 `internal/githubdownload` 候选 URL 生成包，复用到桌面更新和 COLO 词典下载；COLO locations/country 默认源切到 GitHub raw，并补充候选链、去重和失败路径测试。
- 桌面、WebUI 和 Android 的输入源下载默认禁用环境代理，避免系统代理或环境变量影响测速/来源读取行为。
- Dashboard 进度区、任务操作按钮、活动列表和 Sources 输入源表单继续优化移动端布局、暗色模式对比度和文本容纳能力。
- 新增快速上手文档，补充导出目录、COLO 词典、输入源和测速流程；README、文档索引、Android 文档、部署文档和产品介绍同步 pnpm 与更新链路说明。
- `scripts/release-preflight.sh`、quality workflow、doctor/audit/check/lint/format/verify 脚本同步 pnpm lock、Capacitor 8.4.0 锁文件检查和 Android 发布基线。
- GitHub Release、Android resubmit、GHCR / Docker、Wails、Go runtime 和 Android 默认版本号统一提升到 `1.8.4`，Android 默认 `versionCode` 提升到 `10804`。
