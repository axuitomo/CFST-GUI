# CFST-GUI v1.7.1

`v1.7.1` 主要聚焦 Android 启动探测闪退修复、在线更新 APK 选择链路加固，以及发布面版本同步。

## 重点更新

- 修复 Android 14+ 上点击“开始探测”后，前台服务因 `dataSync` 权限/类型不完整而触发闪退的问题。
- Android 前台服务启动新增兼容保护；即使系统拒绝拉起前台服务，也会回传明确失败事件，而不是直接把进程打崩。
- 在线更新下载链路改为客户端按优先级尝试：`ghproxy.com` 优先，`kkgithub.com` 备选，原始 GitHub Release 地址兜底。
- Android Release 继续保持 3 个 APK 资产：
  - `cfst-gui-android-release.apk`：universal 通用包
  - `cfst-gui-android-arm64-v8a-release.apk`
  - `cfst-gui-android-armeabi-v7a-release.apk`
- Android 在线更新会优先按设备 ABI 选择更匹配的远程 APK，旧版本客户端仍可回退到 universal 包。
- GitHub Release、GHCR / Docker、Android 默认版本号统一提升到 `1.7.1`，Android 默认 `versionCode` 提升到 `10701`。
