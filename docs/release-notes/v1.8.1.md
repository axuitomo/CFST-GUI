# CFST-GUI v1.8.1

## 变更摘要

- 全端应用图标替换为蓝底白色闪电图标，覆盖桌面主图标、托盘图标、Windows 安装器图标、前端 favicon 和 Android 启动器图标。
- Android adaptive icon 前景层改为多密度 PNG 资源，保留现有 `@mipmap/ic_launcher` / `@mipmap/ic_launcher_round` 引用。
- Android themed icon / monochrome 图层改为透明背景的白色闪电遮罩，开启系统主题图标时只对闪电形状染色，避免整块背景被主题色覆盖。
- GitHub Release、Android resubmit、GHCR / Docker 和 Android 默认版本号统一提升到 `1.8.1`，Android 默认 `versionCode` 提升到 `10801`。
