# CFST-GUI v1.7.9

## 变更摘要

- 在线更新检查改为只读取 GitHub Releases latest 版本信息，发现新版本后不再在检查阶段拉取更新 manifest 和匹配安装包，降低设置页“检查更新”的等待时间。
- 桌面/WebUI 更新请求固定走 TCP，避免默认 H3/QUIC 探测在部分网络环境下拖到超时。
- 下载更新包改为四源并发竞速：`ghproxy.vip`、`gh.3w.pm`、`gh.ddlc.top` 和原始 GitHub Release 地址同时尝试，优先使用最先完整下载且通过 SHA256 校验的结果。
- Android 在线更新同步使用四源竞速下载 APK，并修复下载失败错误拼接对 Android API 23-25 的兼容性。
- README、Android、Docker/环境变量和功能接口文档同步更新在线更新下载策略说明。
- GitHub Release、Android resubmit、GHCR / Docker 和 Android 默认版本号统一提升到 `1.7.9`，Android 默认 `versionCode` 提升到 `10709`。

## 验证

- `bash scripts/release-preflight.sh 1.7.9 --allow-dirty`
- `go test ./internal/app -count=1`
- `git diff --check`
- 真实网络 smoke：正式 `cfst-gui-linux-amd64.tar.gz` 通过四源竞速下载到 `/tmp`，SHA256 校验通过。
