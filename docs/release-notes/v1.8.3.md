# CFST-GUI v1.8.3

## 变更摘要

- 前端工具链同步到 Vite 8、Tailwind CSS 4、TypeScript 6、`@vitejs/plugin-vue` 6 和 `vue-tsc` 3；Tailwind 改由 `@tailwindcss/vite` 接入，PostCSS 配置只保留 Autoprefixer，并刷新 `frontend/dist` hashed assets。
- 桌面端和移动端布局继续收口：主壳、Dashboard、DNS、Workflow、Results 和 Pipeline Studio 视图统一使用新的桥接值处理与响应式布局细节。
- Pipeline Studio 目录、节点、检查器和运行结果展示更新，改进运行状态、节点配置和结果字段的前端处理。
- Cloudflare DNS、colo 字典、测速结果、端口来源、CSV 输出和上传选择链路补充测试与边界处理。
- WebUI、桥接配置、更新检查和命令返回结构继续对齐三端行为，减少桌面、WebUI 和 Android 的差异。
- GitHub Release、Android resubmit、GHCR / Docker、Wails 和 Android 默认版本号统一提升到 `1.8.3`，Android 默认 `versionCode` 提升到 `10803`。
