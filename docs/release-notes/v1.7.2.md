# CFST-GUI v1.7.2

`v1.7.2` 主要聚焦桌面端与 WebUI 的异步任务链路收口、配置时间显示与保存体验修复，以及发布面默认版本同步。

## 重点更新

- 后端桌面探测入口补齐真正的 async accepted 模式，并保留旧同步接口兼容，避免桌面端、WebUI 与旧调用方契约漂移。
- 任务快照与结果刷新链路继续加固：accepted 阶段不再误读旧 `result.csv`，任务完成后会更及时刷新当前结果。
- 任务快照持久化新增更安全的 `task_id` 文件名映射，避免异常 ID 直接进入本地路径。
- 桌面端与安卓端的 UTC 偏移配置、主题自动切换时间计算、时间显示格式与配置保存链路进一步对齐。
- Linux ARM 平台在下载协议为 `auto` 时，默认回退到更稳的 TCP 模式，降低 H3/UDP 异常带来的测速波动。
- 在线更新页继续区分 WebUI 手动部署与桌面端手动替换文件两类提示，降低错误操作概率。
- GitHub Release、GHCR / Docker、Android 默认版本号统一提升到 `1.7.2`，Android 默认 `versionCode` 提升到 `10702`。
