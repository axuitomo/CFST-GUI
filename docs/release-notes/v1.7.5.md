# CFST-GUI v1.7.5

## 变更摘要

- 工作流画布补齐输入源组配置：可从当前绑定配置或已有输入组档案选择输入源，并支持“全部启用输入源”和自定义勾选两种模式。
- 工作流运行事件展示更细：状态栏、当前测试进程和运行日志会显示具体目标、具体节点以及分支命中结果。
- 工作流数据预览补充 IP 地址列；桌面浮层和移动端卡片都能直接看到 IP、测速端口、平均速率和最高速率。
- 桌面/WebUI 工作流节点连续性继续收口，默认模板按输入源组、筛选、TCP、追踪、下载、结果检查与输出、结束链路组织。
- Android 调度链路接入 WorkManager，补齐前台服务和移动端配置兼容处理。
- GitHub Release、Android resubmit、GHCR / Docker 和 Android 默认版本号统一提升到 `1.7.5`，Android 默认 `versionCode` 提升到 `10705`。
