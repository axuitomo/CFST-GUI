# CFST-GUI v1.8.5

## 变更摘要

- 设置页与输入源页改为自动保存：交互、切页和关闭页面前会尝试静默保存，输入组切换/删除当前输入组前会先补齐当前变更，减少手动保存丢失配置的概率。
- 调整默认测速参数：下载测速默认时长从 `10` 秒降到 `4` 秒，预热时长从 `5` 秒降到 `1` 秒；CLI `-dt` 默认值同步调整为 `4` 秒，配置快照默认值一并同步。
- 调整默认上传 Top N：Cloudflare 默认 `top_n` 改为 `5`，GitHub 默认 `top_n` 改为 `20`；配置归一化、上传筛选和移动端默认快照同步保持一致，显式填写 `0` 仍表示不限。
- 配置文档、接口文档、部署/环境变量文档和当前版本入口同步更新到 `1.8.5`；Release、Android resubmit、Container workflow、Wails、Go runtime 和 Android 默认版本号统一提升到 `1.8.5`，Android 默认 `versionCode` 提升到 `10805`。
