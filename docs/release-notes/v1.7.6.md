# CFST-GUI v1.7.6

## 变更摘要

- DNS 页面拆分为只读记录查询入口，支持按 Zone、当前配置记录名或自定义名称读取 Cloudflare A/AAAA 记录，避免误触发线上记录修改。
- Cloudflare 推送链路收敛到工作流、定时任务和测速后自动推送，复用共享上传筛选、Top N 和分流规则。
- 新增测速后自动推送配置，可在手动测速完成后按需推送到 Cloudflare DNS 或导出到 GitHub。
- 配置结构补齐顶层 `cloudflare`、`github` 和 `post_probe_push`，同时兼容旧 `export.github` 与旧 `upload.cloudflare` / `upload.github` 配置。
- 当前结果页补充分页、排序、状态筛选和 IPv4/IPv6 筛选，桌面、WebUI 与移动端共享结果处理逻辑。
- 文档同步更新 Cloudflare API Token 权限、配置字段、DNS 读取与推送边界，以及产品功能说明。
- GitHub Release、Android resubmit、GHCR / Docker 和 Android 默认版本号统一提升到 `1.7.6`，Android 默认 `versionCode` 提升到 `10706`。
