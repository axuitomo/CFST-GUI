# CFST-GUI v1.7.8

## 变更摘要

- 移除 Go 1.24+ 下已 deprecated 且无效的 `rand.Seed` 初始化，依赖 Go 1.20+ `math/rand` 顶层随机源自动播种，保持现有 IP 取样逻辑不变。
- WebUI 服务端目录选择弹窗改用 `createElement`、`textContent` 和 `replaceChildren()` 构建 DOM，避免 `innerHTML` 拼接带来的注入风险。
- GitHub 结果导出改为服务端统一应用共享上传筛选和 `top_n`，桌面端与移动端导出口径保持一致。
- 桌面端、移动端探测和同步工作流增加 panic recovery，异常退出时会记录错误、发送失败事件并释放当前任务/管道状态。
- 桌面端和 WebUI 探测事件发送增加 bridge panic 保护，移动端事件 sink 异常不会再打断任务状态持久化。
- 前端启动流程改为逐步容错，单个启动调用失败不会阻塞配置、运行时状态、调度器和工作流结果恢复。
- 修复前端格式检查失败的文件，并刷新 `frontend/dist` 构建产物。
- GitHub Release、Android resubmit、GHCR / Docker 和 Android 默认版本号统一提升到 `1.7.8`，Android 默认 `versionCode` 提升到 `10708`。
