# CFST-GUI v1.8.2

## 变更摘要

- 修复 Android WebView 中输入框被软键盘遮挡的问题：Activity 使用 `adjustResize`，前端按 `visualViewport` 同步可视高度，并在键盘打开时隐藏底部导航。
- 修复 Android 结果页筛选下拉显示为系统白色大面板的问题：移动端结果筛选改为应用内 picker，支持点外关闭、滚动关闭、Esc 关闭和基础 ARIA 状态。
- Android 发布基线同步到 Capacitor `8.4.0`、Cordova Android `15.0.0`、AGP `9.2.0`、Gradle `9.4.1`、SDK platform `android-36`、Build Tools `36.0.0` 和 NDK `26.3.11579264`。
- Android 构建继续强制 JDK 24，并把 Android 子项目 compile options 统一覆盖为 Java 24；发布预检会校验 SDK 36、Cordova Android 15、Capacitor 8.4.0 lockfile 和运行时默认版本。
- GitHub Release、Android resubmit、GHCR / Docker、Wails 和 Android 默认版本号统一提升到 `1.8.2`，Android 默认 `versionCode` 提升到 `10802`。
- 新增 Android splash 多密度 PNG 资源，保持移动端启动资源与发行基线同步。
