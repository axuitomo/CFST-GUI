# CFST-GUI v1.7.7

## 变更摘要

- 工作流运行时按职责拆分 DAG、节点执行、payload 解析和 runtime 状态文件，降低策略管道维护成本。
- 修复工作流目标禁用状态被忽略的问题，禁用 target 会进入既有跳过路径，不再执行探测或投递节点。
- 前端恢复旧版工作流 action alias 兼容，`run_probe`、`dns_push`、`github_export` 等旧配置和历史结果会正确归一化展示。
- 输入源页支持离开页面时自动保存当前输入组和完整配置；桌面端、WebUI / Docker 与移动端共用同一保存链路。
- WebUI / Docker 输入源保存接口完成静态链路和 `webui` build tag 验证，确认 `UpdateCurrentSourceProfile` 与 `SaveDesktopConfig` 可用。
- 工作流视图改由统一 `WorkflowView` 承载，桥接层拆分为 command、config、pipeline 和 types 模块。
- 桌面侧边栏支持折叠并持久化偏好，输入源页移除重复保存按钮，减少“手动保存”和“离开自动保存”的交互冲突。
- GitHub Release、Android resubmit、GHCR / Docker 和 Android 默认版本号统一提升到 `1.7.7`，Android 默认 `versionCode` 提升到 `10707`。
