# CFST-GUI v1.7.4

`v1.7.4` 聚焦发行阻塞收口与三端发布面同步，确保桌面端、Android 手机端、Linux WebUI / Docker Compose bundle 和 GHCR 多架构镜像统一发布到 `1.7.4`。

## 重点更新

- Android 存储目录固定回到 app 私有目录，移除旧版自定义存储目录选择链路；旧配置中的自定义目录只用于一次性迁移，避免 SAF 与运行时镜像目录状态分叉。
- Android 导出目录改为独立 SAF 持久授权，CSV、测速结果和调试日志通过“结果导出”区域统一选择或清除导出目录。
- 设置页补充调试日志导出入口，并调整存储状态、运行时目录和导出目录说明，减少桌面端、WebUI 与 Android 的行为差异。
- Windows 桌面发行资产切换为签名 MSIX：Release workflow 会使用 `CFST_WINDOWS_SIGNING_CERT_BASE64`、`CFST_WINDOWS_SIGNING_PASSWORD` 和 `CFST_WINDOWS_MSIX_PUBLISHER` 打包并签名 `cfst-gui-windows-amd64.msix`。
- GitHub Release、Android resubmit、GHCR / Docker 和 Android 默认版本号统一提升到 `1.7.4`，Android 默认 `versionCode` 提升到 `10704`。
- Linux WebUI 发行包继续包含 Docker Compose 上下文；GHCR workflow 将发布 `ghcr.io/axuitomo/cfst-gui:1.7.4` 与 `ghcr.io/axuitomo/cfst-gui:v1.7.4` 多架构镜像。
