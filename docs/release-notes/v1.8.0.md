# CFST-GUI v1.8.0

## 变更摘要

- 新增桌面、WebUI、Android 共用的运行时清理链路，周期性释放 idle 连接、过期缓存和内存快照，并避免在任务运行中执行重清理。
- 桌面和移动端任务状态增加 active runtime / paused runtime / persisted-only 区分，取消、恢复、定时任务重入和后台服务收尾更稳定。
- WebUI 结果接口补充来源类型和路径信息，当前结果页可从持久化结果或 CSV 回填，运行时诊断接口支持受控开启。
- Android 默认最低支持版本提升到 API 26，前台服务、通知权限、SAF 文件选择、后台定时任务和 APK 更新下载链路同步整理。
- Android lint 和资源清理完成：迁移 adaptive icon 资源、补充 monochrome 图层、整理 splash fallback，并移除旧密度图重复资源。
- 前端 bridge 类型和 UI 状态同步更新，修复通知权限类型导出和运行时/结果状态展示。
- GitHub Release、Android resubmit、GHCR / Docker 和 Android 默认版本号统一提升到 `1.8.0`，Android 默认 `versionCode` 提升到 `10800`。
